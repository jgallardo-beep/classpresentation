# Class Presentation Hub

A simple GitHub Pages landing page for browser-based HTML presentations.

## Included files

- `index.html` - landing page
- `week2-session1-information-systems.html` - student-facing lesson

The lesson version has teacher scripts and answer-key slides removed.

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder.
5. Click **Save**.

Your website should use this address:

`https://jgallardo-beep.github.io/classpresentation/`

## Add to Google Classroom

Create a **Material** or **Assignment**, then paste the GitHub Pages address. Students can open the lesson from the landing page without downloading a PowerPoint file.
